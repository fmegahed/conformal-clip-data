# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-11-06

### Added
- Comprehensive README improvements highlighting the package as a companion to clip-conformal
- Installation instructions with pip and source installation examples
- Quick start code example for accessing datasets
- Dedicated section explaining relationship with clip-conformal package
- Proper BibTeX citations for the research paper and spc4sts package
- GitHub Actions workflows for automated testing and PyPI publishing
- Test workflow for Python 3.9, 3.10, 3.11, and 3.12
- Release instructions document (RELEASE.md)
- Comprehensive .gitignore file for Python package development

### Changed
- Converted dataset summary table to proper Markdown format for better rendering
- Improved README structure with clearer sections and navigation
- Enhanced package positioning as a data-only companion package

### Fixed
- Corrected author list in citation to match the actual arXiv paper

## [0.1.0] - 2025-11-03

### Added
- Initial release of the textile image dataset
- 1,000 nominal images of standard textile weave patterns
- 500 local defect images with localized disruptions
- 500 global defect images with systematic parameter shifts
- All images at 250 Ã— 250 px resolution
- Dataset generated using spc4sts R package
- MIT License
- Basic README with dataset description
- Package metadata and distribution configuration
